# SnapShot Pro 📸

A high-performance Chrome extension for capturing tabs and manipulating screenshots directly inside an HTML5 Canvas ecosystem. Built completely from scratch with zero framework overhead.

---

## 🚀 Features Fully Implemented (Phase 4)

* **Asynchronous Screenshot Engine**: Communicates with background Service Workers using Chrome API hooks to capture high-definition viewports.
* **Vector Geometry Tools**: Clean mathematical calculations for rendering shapes (Rectangles and Vector Arrows) on mouse tracks without trailing artifact bugs.
* **Advanced Image Modification Layers**:
    * `Pen Tool` for manual markup styling.
    * `Highlight Tool` using specific color opacity structures.
    * `Blur Engine` utilizing contextual memory blocks to obscure sensitive strings/data elements safely.

---

## 🛠️ Tech Stack & Concepts Explored

* **Manifest V3 Extensions Ecosystem**
* **Asynchronous Chrome Runtime messaging channels**
* **HTML5 Canvas API Context Filters & State Management** (`getImageData` / `putImageData` mechanisms)
* **Trigonometric Vector Coordinates Mapping**

## 📥 Local Installation & Verification Setup

To execute and verify this repository environment locally within Google Chrome:

1. Clone or download this repository architecture block onto your workstation workspace.
2. Launch your Google Chrome application instance.
3. Navigate to the extension portal via: `chrome://extensions/`
4. Toggle the **Developer mode** switcher configuration (Top Right) to **ON**.
5. Click the primary **Load unpacked** management button (Top Left).
6. Target and load the base directory folder asset containing this structure.
7. Open the Extension action wrapper viewport and initiate screen manipulation pipelines.
